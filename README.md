<h1>📘 Sistema de Reserva de Prácticas para Autoescuela</h1>

<p>
    Plataforma web desarrollada con <strong>HTML</strong>, <strong>CSS</strong> y <strong>JavaScript</strong> que permite a los alumnos 
    gestionar y reservar sus prácticas de conducción de forma rápida, intuitiva y organizada.
</p>

<hr>

<h2>🚗 Descripción del Proyecto</h2>
<p>
    Este sistema ofrece una interfaz clara y moderna donde los alumnos pueden consultar horarios disponibles, 
    seleccionar fechas y confirmar sus reservas. Está diseñado para ser totalmente responsive y adaptable a 
    distintos dispositivos.
</p>

<hr>

<h2>🛠️ Tecnologías Utilizadas</h2>
<ul>
    <li><strong>HTML5</strong> — Estructura del contenido</li>
    <li><strong>CSS3</strong> — Estilos, diseño y maquetación</li>
    <li><strong>JavaScript</strong> — Lógica de interacción y gestión dinámica de reservas</li>
</ul>

<hr>

<h2>✨ Funcionalidades Principales</h2>
<ul>
    <li>Calendario interactivo para seleccionar prácticas</li>
    <li>Visualización de horarios disponibles</li>
    <li>Confirmación de reserva con validación</li>
    <li>Interfaz responsive para móviles, tablets y ordenadores</li>
    <li>Gestión de datos mediante <em>localStorage</em> (según implementación)</li>
</ul>

<hr>

<h2>📂 Estructura del Proyecto</h2>

<h2>🚀 Cómo Ejecutarlo</h2>
<ol>
    <li>Clona o descarga este repositorio.</li>
    <li>Abre el archivo <strong>index.html</strong> en tu navegador.</li>
    <li>Explora el calendario y realiza reservas.</li>
</ol>
